### Prerequisites
- [Flutter SDK](https://docs.flutter.dev/get-started/install) (latest stable)
- Dart SDK

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/hozi8-web3/devpocket
   ```
2. Navigate to the project directory:
   ```bash
   cd devpocket
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run
   ```
